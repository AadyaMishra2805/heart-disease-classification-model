# heart-disease-classification-model
# Heart Disease Prediction using Machine Learning

## Project Overview
This project predicts the presence of heart disease using machine learning classification algorithms. The workflow includes data preprocessing, exploratory data analysis (EDA), feature scaling, model training, evaluation, and prediction.

---

## Dataset
Dataset used: Heart Disease Dataset from Kaggle

Features include:
- Age
- Sex
- Chest Pain Type
- Cholesterol
- Blood Pressure
- Maximum Heart Rate
- And more...

Target:
- 1 → Heart Disease
- 0 → No Heart Disease
